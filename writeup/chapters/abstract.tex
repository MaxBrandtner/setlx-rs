\begin{abstract}

\renewcommand{\abstractname}{Abstract}

SetlX is an imperative programming language designed to closely mirror the notation of mathematics and set theory, drawing inspiration from SETL, one of the first languages to offer sets and maps as first-class data types. This makes SetlX a natural fit for teaching and prototyping algorithms in an academic context. The reference implementation is an interpreter written in Java, using ANTLR4 as its parser generator. This project reimplements SetlX as an interpreter in Rust, transforming the ANTLR4-based ALL(*) grammar into a LALR(1) grammar using LALRPOP, and constructing an appropriate interpreter pipeline.

\end{abstract}

\begin{abstract}

\renewcommand{\abstractname}{Abstract}

\par
SetlX ist eine imperative Programmiersprache, die darauf ausgelegt ist, die Notation der Mathematik und Mengenlehre möglichst direkt abzubilden. Sie lehnt sich an SETL an, eine der ersten Programmiersprachen, die Mengen und Abbildungen als erstklassige Datentypen unterstützt. Dies macht SetlX zu einem natürlichen Werkzeug für die Vermittlung und Entwicklung von Algorithmen im akademischen Kontext. Die Referenzimplementierung ist ein in Java geschriebener Interpreter, der ANTLR4 als Parser-Generator verwendet. Dieses Projekt reimplementiert SetlX als Interpreter in Rust, transformiert die ANTLR4-basierte ALL(*)-Grammatik in eine LALR(1)-Grammatik unter Verwendung von LALRPOP und konstruiert eine passende Interpreter-Pipeline.

\end{abstract}
